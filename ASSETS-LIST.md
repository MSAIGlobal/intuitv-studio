All assets should be delivered as:
- **Images:** PNG with transparency where appropriate
- **Videos:** MP4, H.264 codec, 30-60fps
- **Resolution:** As specified above or higher
- **Naming:** Exactly as listed in filenames
- **Location:** `/public/` directory in Next.js project
